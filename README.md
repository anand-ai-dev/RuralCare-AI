# 🏥 RuralCare AI

AI-Powered Rural Healthcare Triage Assistant built using React, FastAPI, Groq Llama 3.1, and Retrieval-Augmented Generation (RAG).

## Features
- AI-powered symptom analysis
- RAG-based medical knowledge retrieval
- Risk assessment (Critical, High, Moderate, Low)
- First-aid recommendations
- PDF report generation
- Responsive UI

## Tech Stack
- React + Vite
- FastAPI
- Python
- Groq API
- Llama 3.1
- RAG
- ReactMarkdown
- jsPDF
